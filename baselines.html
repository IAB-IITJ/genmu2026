Task 1 release setup — Face Unlearning Track

Datasets

CelebA and FFHQ

Target identity samples are drawn from CelebA (202,599 celebrity face images) and FFHQ (70,000 high-quality face images). The task specifies three identities per evaluation instance. ArcFace embeddings are used for similarity, retention, and leakage analysis.

CelebA ↗ FFHQ ↗ Download release packet ↓
Base model

Arc2Face

Arc2Face is a face-identity-conditioned image generator built on top of Stable Diffusion, where identity is controlled via ArcFace embeddings. It is a natural fit for identity-centric forgetting tasks because the identity representation is explicit and separable from style.

Arc2Face on HuggingFace ↗ GitHub ↗

Task 2 release setup — Speech Unlearning Track

Datasets

Target-speaker speech bundle

Release packets include target-speaker speech samples and a retention set covering diverse prosody, accents, and emotional styles. Candidate corpora include LibriTTS, VCTK, and LJ Speech — the exact corpus configuration will be distributed with the release packet.

LibriTTS ↗ VCTK ↗ LJ Speech ↗ Download release packet ↓
Base model

StyleTTS2

StyleTTS2 is a high-quality TTS model that disentangles speaker style from linguistic content, making it well-suited for speaker identity unlearning. The evaluation also uses ECAPA-TDNN embeddings for speaker verification and Whisper for intelligibility (WER).
