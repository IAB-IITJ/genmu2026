Track A release setup

Dataset

Samples selected from CelebA and FFHQ. The task specifies three identities per task and uses ArcFace embeddings for similarity and retention analysis.

Model

Arc2Face is listed as the base model for face identity unlearning. It is a natural fit for the identity-centric forgetting tasks introduced in the doc.

Track B release setup

Dataset

Release packets should include target-speaker speech samples and a retention set with diverse prosody, accents, and emotional styles. The challenge wording leaves the exact corpus open, so the site labels this as a speech-release bundle rather than a single fixed corpus.

Model

The document describes the base system generically as a pretrained text-to-speech or voice-cloning model M, which keeps the benchmark flexible across current speech generators.
